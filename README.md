========
ansible-playbooks for personal PC setup

# How to run
Clone from github

	mkdir ~/Development/
	cd Development
	git clone git@github.com:assout/ansible-playbooks.git

install ansible

	sudo yum install ansible

run ansible

	ansible-playbook main.yml -Kv
